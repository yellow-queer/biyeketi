## 🔗 详细依赖关系

### 1. 配置层
**config.py**
- 被所有训练脚本导入
- 定义全局超参数
- 定义数据集路径
- 定义模型配置

```
依赖：无
被依赖：main.py, train_comparison_models.py, compare_backbones.py, 
       run_all_experiments.py, utils.py, models/*
```

---

### 2. 基础模块层

#### utils.py
- 数据集类定义
- 训练/评估函数
- 可视化函数
- 指标计算

```
依赖：config.py, torch, numpy, matplotlib, sklearn
被依赖：main.py, train_comparison_models.py, visualize_predictions.py
```

#### data_preprocess/image_preprocess.py
- 视频抽帧
- 图像增强
- 数据预处理

```
依赖：cv2, os, shutil
被依赖：手动运行或数据集准备脚本
```

#### models/multi_view_model.py
- MultiViewAttentionFusionModel 类

```
依赖：torch, torch.nn, config.py
被依赖：main.py
```

#### models/comparison_models.py
- SingleViewModel
- SimpleConcatModel
- CNNLSTMModel
- CNNLSTMAttentionModel

```
依赖：torch, torch.nn, config.py
被依赖：train_comparison_models.py
```

### 3. 训练脚本层

##  典型执行流程

### 训练流程
```
1. 用户运行：python main.py
   ↓
2. 导入：config.py（加载配置）
   ↓
3. 导入：utils.py（加载工具函数）
   ↓
4. 导入：models/multi_view_model.py（加载模型）
   ↓
5. 创建数据集：utils.MultiViewDataset()
   ↓
6. 训练模型：utils.train_one_epoch()
   ↓
7. 保存结果：checkpoints/attention_fusion/
```

### 可视化流程
```
1. 用户运行：python visualize_predictions.py
   ↓
2. 加载模型：checkpoints/xxx/best_model.pth
   ↓
3. 加载样本：datasets/test/
   ↓
4. 生成热力图：Grad-CAM
   ↓
5. 保存图片：shown/heatmaps/
```

### 网站启动流程
```
1. 用户运行：python app.py
   ↓
2. 导入：config.py（网站配置）
   ↓
3. 注册 Blueprint：api/*
   ↓
4. 初始化技能：skills/*
   ↓
5. 启动 Flask 服务器
   ↓
6. 访问：http://127.0.0.1:5000
```
