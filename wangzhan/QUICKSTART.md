## 🎯 系统简介

### 核心功能
- **📸 图像检测**：上传柑橘图像，使用 ConvNeXt-Tiny 模型进行实蝇侵染检测
- **📊 质量评估**：自动评估图像质量（清晰度、亮度、对比度、色彩），提供专业改进建议
- **💬 智能对话**：基于 Qwen-plus 的多智能体对话，解读检测结果、回答专业问题
- **🔍 联网搜索**：获取最新的柑橘实蝇防治信息
- **📚 知识库检索**：基于本地 RAG 系统，检索专业知识文档（支持 PDF、Word、JPG 等格式）

### 技术优势
- ✅ 深度学习模型：ConvNeXt-Tiny，准确率高
- ✅ 多智能体系统：专业解读 + 防治建议
- ✅ 图像质量评估：PSNR、SSIM、MSE 等专业指标
- ✅ 跨平台访问：本地、局域网、公网全支持

## 🌍 公网访问配置

### 方案一：Ngrok（推荐⭐⭐⭐⭐⭐）

#### 优点
- ✅ 配置简单，3 分钟搞定
- ✅ 免费使用
- ✅ 自动 HTTPS
- ✅ 无需自己的服务器

#### 配置步骤

**步骤 1：注册账号**
- 访问：https://ngrok.com/
- 点击 "Sign Up" 注册账号
- 验证邮箱

**步骤 2：下载客户端**
- 访问：https://ngrok.com/download
- 下载对应系统的客户端
- 解压到任意目录

**步骤 3：配置认证令牌（可选，提高限额）**

- 运行：`ngrok config add-authtoken YOUR_TOKEN`

**步骤 4：启动 Ngrok**
```bash

ngrok http 5000

# 方式 2：使用 PowerShell 脚本
.\start_public.ps1
```

**步骤 5：获取公网 URL**
```
Forwarding: https://xxxx-xxxx-xxxx.ngrok-free.app -> http://localhost:5000
```

**步骤 6：访问网站**
- 复制 ngrok 显示的 URL
- 在手机或任何设备上访问

### 方案三：Frp 内网穿透（专业部署）

#### 优点
- ✅ 完全控制
- ✅ 无流量限制
- ✅ 高性能

#### 缺点
- ❌ 需要有自己的服务器
- ❌ 配置复杂

#### 配置步骤

**步骤 1：准备服务器**
- 购买云服务器（阿里云/腾讯云）
- 确保有公网 IP

**步骤 2：配置 frps（服务器端）**
```ini
# frps.ini
[common]
bind_port = 7000
dashboard_port = 7500
token = your_secret_token
```

**步骤 3：配置 frpc（客户端）**
```ini
# frpc.ini
[common]
server_addr = your_server_ip
server_port = 7000
token = your_secret_token

[citrus-web]
type = http
local_port = 5000
custom_domains = citrus.yourdomain.com
```

**步骤 4：启动**
```bash
# 服务器端
./frps -c frps.ini

# 客户端
./frpc -c frpc.ini
```

## 📸 图像检测使用指南

### 步骤 1：上传图像
- 点击上传区域或拖拽图像
- 支持格式：JPG、PNG、BMP、WebP
- 文件大小：< 10MB

### 步骤 2：质量评估（自动）
系统自动评估图像质量：
- **清晰度**：拉普拉斯方差检测
- **亮度**：平均灰度值分析
- **对比度**：标准差评估
- **色彩**：色彩丰富度分析
- **噪声**：局部方差估计

**质量评分标准**：
- 90-100 分：优秀 ✅
- 70-89 分：良好 ✅
- 50-69 分：一般 ⚠️
- < 50 分：较差 ❌

### 步骤 3：开始检测
- 点击 "开始智能检测" 按钮
- 等待 1-3 秒
- 查看检测结果

### 步骤 4：查看结果
- **检测类别**：健康 / 患虫
- **置信度**：模型预测的可信度
- **概率分布**：健康概率 vs 患虫概率

### 步骤 5：智能对话
- 在对话框输入问题
- 例如："这个严重吗？需要什么农药？"
- 获取专业建议

## ❓ 常见问题

### Q1: 图像上传失败
**A**: 检查以下几点：
1. 文件格式是否支持（JPG/PNG/BMP/WebP）
2. 文件大小是否超过 10MB
3. 网络连接是否正常
4. 刷新页面重试

### Q2: 检测结果不准确
**A**: 可能原因：
1. 图像质量不佳（查看质量评分）
2. 柑橘未置于图像中心
3. 光线过暗或过曝
4. 重新拍摄并上传

### Q3: 移动端无法访问
**A**: 解决方案：
1. 确认手机和电脑在同一 WiFi（局域网）
2. 检查防火墙设置
3. 使用 Ngrok 公网访问
4. 清除浏览器缓存

### Q4: 质量评分低怎么办
**A**: 改进建议：
1. **清晰度低**：重新对焦，使用三脚架
2. **亮度过暗**：增加光线或使用闪光灯
3. **对比度不足**：调整拍摄角度，侧光拍摄
4. **噪声大**：降低 ISO 值，后期降噪

### Q5: 如何添加专业知识
**A**: 
1. 准备文档（TXT/PDF/Word）
2. 放入 `RAG\wenjian\` 目录
3. 重启服务器
4. 对话中自动使用
